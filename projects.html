Process

SG processes a large volume of transactions daily, which must be continuously monitored to comply with regulations regarding sanctioned individuals and entities. Analysts are required to review every transaction that are flagged(by Quality Assesment platform) as potential match to a sanctioned person or entity - a labour-intensive and time consuming process. To streamline this workflow and improve operational efficiency, SG wanted to develope a model to screen transaction alerts and forward only those with a high likelihood of being true matches to the analysts for further review. SG also wanted to ensure the ongoing reliability, accuracy of the system by continously tracking data quality, operational performance.

Approach

Adapted and trained models for two distinct client types: Individuals(sanctioned persons) and Corporates(sanctioned entities).
- Individuals:Employed a rule-based approach combined with text similarity metrics (specifically Jaro distance) to assess the resemblance between client names and names on sanction lists. The model calculates edit distances and applies thresholds to determine if an alert is a potential match. - Corporates: Developed a machine learning model using the Extra Trees classifier. Employed oversampling for "True" labeled alerts to address class imbalance. Conducted hyperparameter tuning(grid search over number of estimators, max depth etc) to optimize model precision and finalized the best performing model. - Model Monitoring: Implemented automated routines to monitor missing values in critical alert fields like alert IDs, date of birth of the party, daily alert volume, and data consistency. USed statistical methods like interquartile range,z score to set upper threshold and flag unexpected spikes or drops in alert volumes.

Solution

- The solution was an integrated alert screening system that automatically evaluates daily transaction alerts using both rule-based and machine learning methodologies. The monitoring framework provided robust, automated oversight of the QATP alert screening system.
